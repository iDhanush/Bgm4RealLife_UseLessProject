# Installation
<h4>Backend</h4>
git clone https://github.com/iDhanush/Bgm4RealLife_UseLessProject.git <br>
cd Bgm4RealLife_UseLessProject <br>
pip install -r requirements.txt <br>
add .env file <br>
GEMINI_API_KEY=XXX <br>
BASE_URL=HTTPS://BASEURL/ <br>
# Run<br>
uvicorn main:app<br>
<BR>
<BR>

<h4>frontEnd</h4>
cd frontend <br>
npm i
npm run dev
